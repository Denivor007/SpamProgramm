ПРО ПРОГРАММУ:
программа позволяет рассылать в телеграмме сообщения текстового содержания с форматированием, или массово приглашать людей в каналы.
Вероятно, недостатком программы является то, что она добавляет каждый валидный номер в контакты, что захламит ваш список контактов.
Очистка от этих контактов будет предусмотрена в следующей версии программы.
Если не указан иной файл, данные для рассылки и приглашений будут браться из базы данных, указанной в файле config\\config.txt
(если пользователь не добавлен в список контактов, то вы не сможете ему написать)
в БД(Базе Данных) есть поля:
 id - обычный идентификатор
 user_number - номер телефона
 user_id - айди пользователя, если он зарегестрирован в телеграмм
 active - показатель того, активен ли пользователь. При неудачной отправке сообщения,
  или при условии, что у номера отсутствует ТГ аккаунт, присваивается статус 0.

функции (команды):
1. spam - рассылка сообщений из файла config\\messages.txt
сообщения разделяются тремя дефисами "---"
сообщения выбираются рандомно среди записанных в файл messages.txt
при желании можно установить сообщение при запуске команды spam через параметр -m или --message
2. invite - приглашение людей
обязательно указывайте параметром ссылку на канал
3. parse - из вашего (пока-что только с расширением csv) файла, указанного в параметре -f или --file по ключу,
указанному в параметре -k или --key будут добавленны в БД номера.
Если у номера есть аккаунт в ТГ, то у него появится статус active 1, присвоится user_id и он будет добавлен в ваш список контактов
4. fill - лишний раз перебирает БД, и если у номера есть аккаунт в ТГ, то у него появится статус active 1, присвоится user_id и он будет добавлен в ваш список контактов

ИСПОЛЬЗОВАНИЕ:
(!!!!)
обязательно перед началом работы с программой, используя команду python main.py parse добавьте в базу данных номера телефонов из нужного файла.

инструкция:
войдя в папку с программой нажимаем shift+ПКМ и выбираем открыть командную строку сдесь.
чтоб воспользоваться программой нужно ввести команду со всеми необходимыми настройками.
все команды начинаются на python main.py

команды имеют ряд настроек, которые добавляются после неё.
Например, если я хочу задать задержку между сообщениями в 5 секунд я пишу:
"python main.py spam -d 5" или "python main.py spam --delay 5"
соответственно - пишется настройка, после неё значение. Их может быть много на всю команду.
например
python main.py invite -d 10 -f numbers.csv -k number
приглашает всех пользователей из файла "numbers.csv" по ключу "number" с интервалом в 10 секунд

python main.py spam
'-f' или '--file' задаёт файл, из которого будут браться номера для рассылки. по умлчанию это database из файла setup.py
'-k' или '--key', если файл задан как csv то этот параметр определяет столбец, по которому беуться номера. по умолчанию "number" (в будущем значение по умолчанию будте браться из файла config.txt)
'-d' или '--delay', задаёт задержку отправки сообщения, по умолчанию 10  (в будущем значение по умолчанию будте браться из файла config.txt)
'-m' или '--message', сообщение для рассылки. по умолчанию данные из файла "message.txt"(в будущем значение по умолчанию будте браться из файла config.txt)
'-st' или '--start', номер человека в файле, по умолчанию "0", т.е. первый (в будущем значение по умолчанию будте браться из файла config.txt)
'-c' или '--count', количество номеров для обработки, по умолчанию это максимум.
'-rm' или '--random_mode', когда этот параметр имеет значение "1", то сообщения отправляются в случайном диапазоне,
с центром, выбранным параметром delay. Если delay = 10, то диапазон будет от 5 до 15. по умолчанию '1'
'-am' или '--active_mode', когда этот параметр имеет значение "1", рассылка происходит только по номерам, со статусом active = 1. по умолчанию '1'
